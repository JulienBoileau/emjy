# Emjy

Site public EMJY Production construit avec React, Vite et Firebase.

## Development server

Pour lancer la demo en local :

```bash
npm start
```

Le site est ensuite disponible sur l'URL affichee par Vite, en general `http://localhost:5173/`.

## Build

Pour produire le build de production :

```bash
npm run build
```

Les fichiers generes sont ecrits dans `dist/`.

## Tests

La commande disponible est :

```bash
npm test
```

Elle ne lance pas encore de vraie suite automatisee et retourne actuellement `No tests configured`.

## Stack

- React 18
- Vite 5
- React Router
- Firebase
